# Boston Housing Data Analysis

This project is done for the study of the Boston Housing Data. The data is taken from the Carnegie Melon University's Statistic Department Website. The data is used to predict the median value of owner-occupied homes in $1000's. The data set consists of 506 instances and 14 attributes including the target attribute.

The data is used for exploratory data analysis and has been trained with three different models. Models are linear regression, XGBoost and Random Forest. The models are trained and tested with the data and the results are compared. The best model is selected based on the results.
